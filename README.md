# 🎮 CrewAI & Groq - AI Game Designer Agent

An advanced AI Agent built using **CrewAI** and **Groq (Llama-3.3-70b-versatile)** that takes user concepts/requests and automatically designs highly detailed, mechanically sound **Game Design Documents (GDD)**.

---

## 🚀 Features
- **Llama 3.3 Integration:** Powered by Groq's ultra-fast API infrastructure.
- **Custom Compatibility Patch:** Implements a dynamic `litellm` patch to safely strip incompatible cache fields, preventing API-level crashes.
- **Dynamic User Inputs:** Prompts the user directly via terminal to capture unique gameplay visions.
- **Comprehensive GDD Output:** Generates core loops, narrative structures, and character mechanics effortlessly.
